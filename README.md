# IceNote 📝

A minimalist note-taking application inspired by the simplicity of Google Keep and the privacy-focused philosophy of Obsidian.

## 💡 Project Philosophy
The main goal is to create a fast and intuitive tool where notes belong solely to the user. All data is stored locally on your device, ensuring maximum confidentiality (**Local-first**). No clouds, no tracking, no third-party servers.

## 🚀 Current Features
* **Full Privacy**: Your data never leaves your browser.
* **Quick Thought Capture**: A clean, distraction-free interface.
* **Dynamic Card Generation**: Notes are created on the fly using Vanilla JavaScript.
* **Adaptive Grid Layout**: Smart distribution of notes using CSS Grid.
* **Attention to Detail**:
    * Smooth animations with optimized `transitions`.
    * UI stability: Using `outline` instead of `border` to prevent layout shifts.
    * Interactive removal with custom SVG icons.
## ⚙️ How to Run
Since this is a client-side application, you don't need to install any dependencies:
1. Clone the repository or download the source code.
2. Open `index.html` in any modern web browser.
*Recommended: Use "Live Server" extension in VS Code for the best development experience.*
## 🛠 Tech Stack
* **HTML5 / CSS3**: Advanced layouts with Grid & Flexbox.
* **Vanilla JavaScript**: DOM API, Event Delegation, and Propagation control.
* **SVG Graphics**: Lightweight, built-in icons for instant loading.
